
# Instabot #

**Automatically follow and post to Instagram without official API**

## Prerequisites
 * Python 2.7
 * Click: http://click.pocoo.org/
    * ```pip install click```


## Usage
 * clone repo ```git clone https://github.com/rhawiz/instabot.git```
 * run ```python instabot.py --account <account_id>```
